Record a terminal session as an asciicast

Usage: wakterm record [OPTIONS] [PROG]...

Arguments:
  [PROG]...  Start prog instead of the default_prog defined by your wakterm
             configuration

Options:
      --cwd <CWD>  Start in the specified directory, instead of the default_cwd
                   defined by your wakterm configuration
  -o <OUTFILE>     Save asciicast to the specified file, instead of using a
                   random file name in the temp directory
  -h, --help       Print help
