# `show_tab_index_in_tab_bar = false`

When set to `true`, tab titles show their tab number (tab index) with a
prefix such as `1:`. The default is `false`, so no numeric prefix is shown
unless you opt in.

The [tab_and_split_indices_are_zero_based](tab_and_split_indices_are_zero_based.md) setting controls whether numbering starts with `0` or `1`.
